## [1.0.0] - 2025-11-27

### Added
- Initial release of @youssefmohamed07/chat-widget
- Full TypeScript support with type definitions
- Multilingual support (Arabic and English)
- Responsive chat widget component
- Standalone chat widget component
- Rich UI components:
  - Date range picker
  - Traveler selector
  - Hotel cards
  - Meal plan selector
  - Room type selector
  - Budget range selector
  - Quick replies
  - Booking summary
- Configurable API base URL via props
- Built-in API service with Axios
- Support request functionality
- Customer contact information collection
- Language selection
- Error boundary for error handling
- Event bus system for component communication
- Plugin system for extensibility
- Tailwind CSS-based styling
- ESM and UMD module formats
- Comprehensive documentation
